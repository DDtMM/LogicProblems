# LogicProblems

Class logic puzzles game.  Fun for people ages 8 through 88.

## Features
* Command history
  * Undo
* Multiple sample puzzles
* Puzzle selector
* Puzzle validation
* Reactive UI
* Save game progress.
* Scalable grid
* Themes
